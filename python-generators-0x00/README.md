# 🐍 Python Generators with MySQL Database

This project demonstrates how to combine **Python** and **MySQL** for efficient data handling.  
You will learn how to connect to a MySQL server, create a database and table programmatically, seed the database from a CSV file, and use a **generator** to stream rows from the database in a memory-efficient way.

---

## 🎯 Learning Objectives

By completing this project, you will:

- Understand how to **connect Python to a MySQL database**.
- **Create and manage** databases and tables programmatically.
- **Insert CSV data** into a MySQL table.
- Implement a **generator function** to stream data rows efficiently.
- Learn best practices for **database seeding and resource cleanup**.

---

## ⚙️ Requirements

- Python **3.10+** (Python 3.13 recommended)
- MySQL Server installed locally
- MySQL root access or a user with database creation privileges
- `mysql-connector-python` library installed
- Sample dataset file named **`user_data.csv`**
- (Optional) Git & GitHub for version control
